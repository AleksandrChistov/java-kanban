# java-kanban

Java-приложение для управления задачами по методологии Kanban.

## Описание проекта

Проект представляет собой консольное приложение для управления задачами, которое поддерживает три типа задач:
- Обычные задачи (Task)
- Эпики (Epic) - крупные задачи, состоящие из подзадач
- Подзадачи (Subtask) - элементы, входящие в состав эпика

Приложение реализует основные функции управления задачами: создание, просмотр, обновление и удаление задач всех типов.

## Особенности реализации

### Менеджеры задач

Проект включает два типа менеджеров задач:
- `InMemoryTaskManager` - хранит данные в памяти
- `FileBackedTaskManager` - сохраняет данные в файл (resources/tasks.txt) и может восстанавливать их при перезапуске

### История просмотров

Реализован механизм истории просмотров задач с использованием `InMemoryHistoryManager`, который хранит последние 10 просмотренных задач.

### HTTP API

Приложение предоставляет HTTP API для удаленного доступа к данным через `HttpTaskServer`, который запускается на порту 8080 и предоставляет следующие эндпоинты:
- `/tasks` - управление обычными задачами
- `/subtasks` - управление подзадачами
- `/epics` - управление эпиками
- `/history` - получение истории просмотров
- `/prioritized` - получение задач, отсортированных по приоритету (времени выполнения)

## Использование

Для запуска сервера:
```java
FileBackedTaskManager fileBackedTaskManager = Managers.getDefault();
HttpTaskServer taskServer = new HttpTaskServer(fileBackedTaskManager);
taskServer.start();
```

Сервер будет доступен по адресу http://localhost:8080

## Структура проекта

Проект организован по принципам ООП с разделением на пакеты по функциональности:
- `tasks` - классы задач
- `managers` - менеджеры задач и истории
- `http` - реализация HTTP сервера и обработчиков
- `enums` - перечисления
- `interfaces` - интерфейсы
- `utils` - утилиты
